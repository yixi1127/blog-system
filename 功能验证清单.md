# 博客文章管理系统 - 功能验证清单

## 启动项目

```bash
# 1. 安装依赖（如果还没安装）
npm install

# 2. 启动开发服务器
npm run serve

# 3. 打开浏览器访问
# http://localhost:8080
```

## 常见问题排查

### 问题 1：页面空白
**解决方案：**
- 检查浏览器控制台是否有错误
- 确认 npm run serve 正常运行
- 尝试清除浏览器缓存后刷新

### 问题 2：数据不显示
**解决方案：**
- 打开开发者工具检查 localStorage
- 确认 `blog_articles` 等键存在
- 尝试清除 localStorage 后刷新页面重新初始化

### 问题 3：创建文章失败
**解决方案：**
- 检查必填字段是否都已填写
- 查看浏览器控制台错误信息
- 确认分类已存在

### 问题 4：数据丢失
**解决方案：**
- 检查是否清除了浏览器缓存
- localStorage 数据仅存储在当前浏览器
- 建议定期备份重要数据
