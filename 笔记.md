## layout 全局通用布局,使得每个页面都确定同一个底部和顶部

### 底部栏

#### 样式向下居中

<style>
#app .footer {

background-color: #efefef;
  text-align: center;
  position: fixed;
  bottom: 0;
  left: 0;
  right: 0;
padding: 16px;//边距
}
</style>
#### ctrl + alt + l 格式化代码
#### 保持底部不变(动态的内容替换)       
<a-layout-content>
<router-view />
</a-layout-content>

#### 保证内容不会被tun
#app .container {
padding: 20px;
margin-bottom: 20px;
background: linear-gradient(to right, #efefef, #fff);
}(因为底部栏16px,保留20px的空余位置,那么肯定是不会被吞掉的)

#### css
app.header {
background: white;        /* 白色背景 */
margin-bottom: 16px;      /* 与下方元素间距 16px */
color: unset;             /* 文字颜色继承 */
padding-inline: 20px;     /* 左右内边距各 20px */
}
1. padding-inline: 20px;
   等于：左右内边距各 20px
   效果：元素内容与边框之间的左右间距
2. margin-bottom: 16px;
   等于：底部外边距 16px
   效果：元素与下方其他元素之间的间距
