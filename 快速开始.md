# 🚀 快速开始指南

## 一分钟启动项目

### 1️⃣ 安装依赖（首次运行）
```bash
npm install
```

### 2️⃣ 启动开发服务器
```bash
npm run serve
```

### 3️⃣ 打开浏览器
访问：http://localhost:8080

---

## 🎯 快速体验功能

### 创建第一篇文章
1. 点击导航栏 **"文章列表"**
2. 点击 **"新建文章"** 按钮
3. 填写信息：
   - 标题：`我的第一篇文章`
   - 分类：选择 `技术文章`
   - 标签：输入 `测试` 并按回车
   - 内容：`这是我的第一篇文章内容...`
4. 选择 **"立即发布"**
5. 点击 **"发布文章"**

✅ 完成！你的文章已经真实保存到浏览器 localStorage 中了！

### 验证数据持久化
1. 按 **F5** 刷新页面
2. 查看文章列表
3. 你的文章还在！✨

## 💡 快速提示

### 查看存储的数据
1. 按 **F12** 打开开发者工具
2. 进入 **Application** → **Local Storage**
3. 查看 `blog_articles`、`blog_categories` 等键

### 清除所有数据
在开发者工具的 Console 中执行：
```javascript
localStorage.clear();
location.reload();
```

### 默认数据
系统会自动初始化：
- 3 个分类（技术文章、生活随笔、学习笔记）
- 2 篇示例文章
- 1 个默认用户（张三）

## ❓ 遇到问题？

### 端口被占用
修改 `vue.config.js`：
```javascript
module.exports = {
  devServer: {
    port: 8081  // 改成其他端口
  }
}
```

### 依赖安装失败
尝试：
```bash
npm cache clean --force
npm install
```

### 页面空白
1. 检查浏览器控制台错误
2. 确认 `npm run serve` 正常运行
3. 尝试清除浏览器缓存
